# 화면 구조 및 UI 설계

## 앱 구조 개요

역할별로 다른 화면 세트를 보여주는 단일 앱 구조

```
앱 시작
    │
    ├── 로그인/회원가입 (공통)
    │
    └── 역할별 메인 화면
        ├── 마스터 앱 (Tab Navigation)
        ├── 관리자 앱 (Drawer + Tab)
        └── 클라이언트 앱 (Tab Navigation)
```

---

## 1. 공통 화면

### 1.1 스플래시 화면
- 앱 로고
- 로딩 인디케이터

### 1.2 로그인 화면
```
┌─────────────────────────────┐
│                             │
│         [앱 로고]           │
│                             │
│  ┌───────────────────────┐  │
│  │ 이메일                │  │
│  └───────────────────────┘  │
│  ┌───────────────────────┐  │
│  │ 비밀번호              │  │
│  └───────────────────────┘  │
│                             │
│  [      로그인 버튼      ]  │
│                             │
│  비밀번호 찾기 | 회원가입   │
│                             │
└─────────────────────────────┘
```

### 1.3 회원가입 화면
- 이메일 입력
- 비밀번호 설정
- 이름 입력
- 전화번호 입력
- 역할 선택 (마스터/클라이언트)
  - 관리자 역할은 기존 관리자가 부여

```
Text:       #1F2937 (진한 회색)
```

- 최소 터치 영역: 44x44px

